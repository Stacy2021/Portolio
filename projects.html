Projects

Maternal and Fetal Monitor

This flagship project involved designing and building a wearable smart belt embedded with multiple biomedical sensors. It monitors both maternal contractions and fetal heart rate, sending real-time data to a cloud-linked dashboard accessible to healthcare workers. The device includes vibration-based feedback for fetal activity simulation and supports community health centers in remote regions by offering mobile-accessible diagnostics. The project addressed SDG 3 (Good Health and Well-being) and was part of my final year capstone.

Smart Home Automation

Built using Arduino Uno, PIR motion sensors, DHT11 temperature sensors, and relay modules, this home automation prototype manages lighting and climate control for efficiency and security. A Blynk app interface was created to enable smartphone-based remote access, while onboard feedback LEDs and a temperature display offered local real-time feedback. The system showcased low-cost solutions for home automation targeted at middle- and low-income homes.

IoT Heartbeat Monitor

This health tech project integrates a pulse sensor with an Arduino board and publishes heartbeat data to ThingSpeak for cloud visualization. Alerts are triggered when abnormal readings are detected. The project emphasizes real-time vital sign tracking in telemedicine contexts, promoting accessibility to health analytics for patients in isolated areas. It also explored Firebase integration for backup storage and remote access through mobile apps.

Accident Prevention System

As part of a workshop on automotive safety, I created a prototype using HC-SR04 ultrasonic sensors mounted on a vehicle model to detect front-side obstacles. When objects are detected within a danger zone, a buzzer and LED indicator alert the driver. The system aims to reduce collision risk in foggy or low-visibility settings and can be expanded with vibration alerts or automatic braking in future iterations.
